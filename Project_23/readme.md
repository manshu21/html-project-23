🚀 Modern SaaS Landing Page

**_About Project_**

This is a simple responsive landing page made using HTML and Tailwind CSS.

I created this project to practice responsive web design. The website works on mobile, tablet and desktop screens.

**_Features_**
1.Responsive navigation bar
2.Mobile hamburger menu
3.Hero section
4.Feature cards
5.Hover effect on feature cards
6.Gradient background
7.Sticky header
8.Responsive footer

**_Technologies Used_**
1.HTML5
2.Tailwind CSS (CDN)
3.JavaScript (for mobile menu)

**_How to Run_**
1.Download or clone the project.
2.Open the project folder.
3.Open index.html in your browser.

That's it.
